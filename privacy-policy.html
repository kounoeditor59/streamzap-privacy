StreamZAP プライバシーポリシー

最終更新日：2025年6月

1. はじめに

StreamZAP（以下「本拡張機能」）は、TwitchおよびYouTubeのフォロー中ライブ配信を一覧表示するChrome拡張機能です。本ポリシーは、本拡張機能がどのような情報を取り扱うかを説明します。

2. 収集する情報

本拡張機能は以下の情報を取り扱います。

  • TwitchおよびYouTubeのOAuthアクセストークン
  • フォロー中チャンネルのID一覧（YouTubeのみ・キャッシュ目的）
  • 拡張機能の設定情報（パネル表示ON/OFF）

3. 情報の保存場所

上記の情報はすべて、お使いのブラウザ内（chrome.storage.local）にのみ保存されます。外部サーバーへの送信・保存は一切行いません。

4. 情報の利用目的

取得した情報は、以下の目的のみに使用します。

  • TwitchおよびYouTube APIへのアクセス（フォロー中のライブ配信情報の取得）
  • 拡張機能の設定の保持

5. 第三者への提供

本拡張機能は、取得した情報を第三者に提供・販売・共有することは一切ありません。

6. 外部サービスについて

本拡張機能はTwitch API・YouTube Data API v3を使用します。これらのサービスのプライバシーポリシーについては、各サービスのポリシーをご確認ください。

  • Twitchプライバシーポリシー
  • Googleプライバシーポリシー

7. YouTube APIサービスについて

本拡張機能はYouTube API Servicesを使用しています。YouTube APIの利用にあたり、本拡張機能は以下に従います。

  • YouTube利用規約
  • YouTubeプライバシーポリシー
  • Googleプライバシーポリシー

8. データの削除

拡張機能の設定画面からアカウント連携を切断することで、保存されたアクセストークンを即時削除できます。また、Chrome拡張機能をアンインストールすることで、すべてのローカルデータが完全に削除されます。データはローカルにのみ保存されるため、サーバー側での削除申請は不要です。

9. お問い合わせ

本ポリシーに関するご質問は、Chrome Web Storeのサポートページよりお問い合わせください。

StreamZAP Privacy Policy

Last updated: June 2025

1. Introduction

StreamZAP (the "Extension") is a Chrome extension that displays live streams from Twitch and YouTube in one place. This policy explains what information the Extension handles.

2. Information We Handle

The Extension handles the following information:

  • OAuth access tokens for Twitch and YouTube
  • Followed channel IDs (YouTube only, for caching purposes)
  • Extension settings (panel display ON/OFF)

3. Data Storage

All of the above information is stored exclusively within your browser (chrome.storage.local). No data is ever transmitted to or stored on external servers.

4. How We Use Information

The information is used solely for the following purposes:

  • Accessing Twitch and YouTube APIs to retrieve live stream information
  • Storing extension settings

5. Third-Party Sharing

The Extension does not share, sell, or provide any collected information to third parties.

6. Third-Party Services

The Extension uses the Twitch API and YouTube Data API v3. Please refer to their respective privacy policies:

  • Twitch Privacy Policy
  • Google Privacy Policy

7. YouTube API Services

This Extension uses YouTube API Services. By using this Extension, you agree to be bound by the following:

  • YouTube Terms of Service
  • YouTube Privacy Policy
  • Google Privacy Policy

8. Data Deletion

You can delete stored access tokens immediately by disconnecting your accounts from the extension's settings popup. Uninstalling the extension will remove all locally stored data completely. Since all data is stored locally, no server-side deletion request is necessary.

9. Contact

For questions about this policy, please contact us through the Chrome Web Store support page.
